## watch

Watch a directory, file, or files for changes and run a command.

### Usage

~~~ bash
$ watch <path> <cmd>...
~~~

### Examples

~~~ bash
$ watch . make
$ watch *.js npm run build
~~~

### Installing

~~~ bash
$ make install
~~~

*Compiling with [Swift version `3.0-dev` (2016-04-25)](https://swift.org/download/#snapshots)*
